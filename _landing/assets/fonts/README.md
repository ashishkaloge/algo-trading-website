# Font Assets

Self-hosted Latin subsets used by the landing page:

- `inter-latin.woff2`
- `plus-jakarta-sans-latin.woff2`

Both are sourced from Google Fonts and distributed under the SIL Open Font
License. They are preloaded by every HTML page, while `font-display: swap` keeps
text readable if a font is not immediately available.

The corresponding license texts are retained at:

- `LICENSES/Inter-OFL-1.1.txt`
- `LICENSES/Plus-Jakarta-Sans-OFL-1.1.txt`
